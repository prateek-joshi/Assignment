# Task 1.1 - Data Visualization

```
python '.\Data Visualization\main.py' -f=<IMAGE_PATH> -j=<JSON_PATH> [-o=<OPACITY>]
```
This creates two new images in the .\images directory with type-1 and type-2 visualizations respectively.
